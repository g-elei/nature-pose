# NaturePose – AI Running Form Coach

Real‑time pose analysis and coaching using MoveNet + TensorFlow.js. Works entirely in the browser – no video leaves your device.

## Features
- Live camera, video upload, or image upload
- Real‑time skeleton overlay and form feedback (voice + text)
- Tracks knee angle, torso lean, heel strike, arm swing, cadence
- Progress dashboard with session history and CSV export
- Nature‑themed UI with full accessibility (ARIA, keyboard)

## Tech Stack
- React 18 + TypeScript + Vite
- Tailwind CSS (custom nature palette)
- TensorFlow.js + MoveNet for pose detection
- IndexedDB for local storage
